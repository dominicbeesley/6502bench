6502bench SourceGen Reference Manual

SourceGen is an interactive disassembler for 6502, 65C02, and 65816 code. The official web site is https://6502bench.com/.

If you want to get up to speed quickly, start with the tutorials.

NOTE: the version of the manual on the web may be older or newer than the version of the program you're using. To ensure you're reading the correct version, open the manual from within SourceGen, using the Help > Help menu item or by hitting F1.
